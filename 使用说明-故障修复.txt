Word Turn 简易说明书（Windows）

一、启动前准备
1) 先安装 Python 3.10 及以上版本。
2) 打开本项目目录，双击 start_word_turn.bat。

二、如果点击“一键修复”后看起来没反应
- 新版本会在后台执行修复，不会让窗口卡死。
- 请观察“系统状态”里的日志：
  - ✅ 表示修复成功
  - ❌ 表示失败（通常是网络或权限问题）

三、常见报错：No module named streamlit
1) 在启动器中先点“选择安装目录”（建议选本项目根目录）。
2) 再点“一键修复”，等待依赖安装完成。
3) 再点“启动 Word Turn”。

四、仍然失败时（手动命令）
在项目目录打开终端，执行：
python -m pip install -r requirements.txt
python -m streamlit run app.py

五、建议
- 使用稳定网络。
- 尽量不要把项目放在需要管理员权限的系统目录。
- 安装成功后，依赖会放在“安装目录\site-packages”。
